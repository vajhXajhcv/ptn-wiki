## 玩法概述

暗域是面向中后期局长的高难挑战，每期会提供不同的场地效果与敌人配置。局长需要根据环境词缀调整阵容，在限定条件下完成通关目标。

## 核心机制

- **环境效果**：每期暗域附带全局增益/减益，影响敌我双方。
- **积分与评级**：根据通关速度、禁闭者存活数、核心击破效率等维度计分。
- **周期轮换**：不同周期开放不同主题与奖励，鼓励尝试多样阵容。
- **难度分级**：通常分为多个难度档位，选择适合自己练度的档位优先拿满奖励。

## 通用配队思路

暗域对队伍完整度要求较高，建议准备以下角色定位：

| 定位 | 作用 | 选择建议 |
|------|------|----------|
| 主 C | 核心输出 | 根据当期敌人抗性选择物理或法术输出 |
| 破核手 | 击破敌人核心 | 至少 1 名，多核敌人可带 2 名 |
| 增益辅助 | 增伤/减抗 | 提升主 C 爆发窗口伤害 |
| 治疗/护盾 | 续航与容错 | 高难环境下保证队伍存活 |
| 坚韧/狂暴 | 承伤与阻挡 | 吸引火力，保护后排输出 |

## 实战技巧

1. **先读环境词缀**：进入战斗前仔细阅读当期 buff/debuff，判断是利好爆发、持续输出还是控制流派。
2. **核心管理**：优先击破精英与 boss 核心，破核后的易伤阶段是主要输出窗口。
3. **移动与站位**：利用局长移动调整角色位置，让主 C 覆盖更多敌人，同时避免脆皮被集火。
4. **能量分配**：把必杀技留给破核后的爆发期，不要在小怪身上过度浪费。
5. **难度选择**：如果无法稳定通关高难度，优先在低难度拿满基础奖励，再逐步挑战更高档位。

## 常见环境词缀应对

- **敌方增伤**：带上护盾或治疗角色提高容错，必要时调整站位分散伤害。
- **我方减能**：选择普攻收益高或自带回能的角色，减少对必杀技的依赖。
- **特定职业强化**：优先使用被强化的职业作为主 C 或辅助。
- **敌人免疫控制**：放弃纯控制流派，改用爆发或持续输出。
