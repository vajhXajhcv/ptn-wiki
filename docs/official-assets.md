# 官网公开资源与合规使用说明

## 资源发现

《无期迷途》国服官网（`https://wqmt.aisnogames.com/`）对外暴露了公开 API：

```
GET https://wqmt.aisnogames.com/api/news?section=1&offset={offset}&limit={limit}
GET https://wqmt.aisnogames.com/api/news/{id}
```

无需登录即可读取。截至本次抓取，共 1611 条资讯，按类型分布如下（仅列出与 Wiki 相关的类别）：

| 类型 | 数量 | 用途 |
|------|------|------|
| 禁闭者档案 | 98 | 角色立绘、设定、异能说明 |
| 禁闭者影像捕获 | 88 | 角色动态展示/GIF、视频 |
| 禁闭者装束 | 159 | 皮肤宣传图 |
| MBCC生日会 | 276 | 角色生日贺图 |
| 壁纸 / 影像壁纸 / 无期记事 | 126+ | 壁纸、角色插画 |
| 预告 / 公告 / 活动资讯 | 若干 | 版本信息、活动说明 |

> 以上数据来自官网公开接口，版权归 **上海自意网络科技有限公司 / 自意网络** 所有。

## 已应用到 Wiki 的资源

脚本 `scripts/fetch-official-resources.mjs` 完成以下工作：

1. 拉取全部资讯列表。
2. 按优先级匹配角色：禁闭者档案 → 禁闭者影像捕获 → 禁闭者装束 → MBCC生日会 → 壁纸。
3. 取第一条静态图片，用 Jimp 压缩为宽 480px、JPEG 质量 85% 的卡片/详情页尺寸。
4. 保存到 `public/characters/{slug}.jpg` 并同步更新对应 Markdown 的 `image` 字段。

本次匹配结果：

- **成功匹配 43/45 名角色**
- 主要来源：MBCC生日会（42 名）+ 禁闭者档案（1 名：罗睺）
- **未匹配**：`EMP`、`K.K.`（官网资讯中暂未出现这两名角色的独立图文）

详细对应关系见 `scripts/tmp/official-character-matches.json`。

## 本地使用

```sh
# 重新拉取并生成图片（会覆盖 public/characters/ 下已有文件）
node scripts/fetch-official-resources.mjs

# 构建站点
npm run build
```

## 合规建议

1. **版权归属**  
   所有图片、文字、角色设定的著作权均归自意网络。本站 Wiki 目前使用官网公开可访问的图片链接/文件，不等同于获得商业或二次传播授权。

2. **使用方式**  
   - 推荐在站点底部或角色详情页增加显著署名，例如：  
     `角色素材来自《无期迷途》官方网站，版权归自意网络所有。`
   - 如官方后续要求下架，应立即移除相关图片。
   - 在取得正式授权前，不建议将图片用于周边售卖、广告投放或其他商业用途。

3. **文字内容**  
   角色正文、攻略为原创撰写，不直接复制 BWiki 或其他社区。如需引用官方设定，建议用自己的语言概括，并标注来源。

4. **后续可替换为更高清/更合适的素材**  
   当前大多数角色使用的是生日贺图。当官网放出对应角色的“禁闭者档案”立绘后，可调整脚本优先级或手动替换为档案立绘。

5. **缺失角色处理**  
   `EMP`、`K.K.` 暂无匹配到的官方图文。可继续观察官方后续资讯，或保留占位图/不显示图片，避免使用非官方来源的素材。
