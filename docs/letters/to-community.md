# 致广大《无期迷途》玩家与创作者的倡议书

**各位局长、攻略作者、画师、译者与数据整理者们：**

大家好！

我们正在维护一个非官方、非营利、完全开源的《无期迷途》玩家 Wiki——**5732.wiki**，希望能为所有喜欢这款游戏的玩家提供一个清晰、准确、长期可用的资料站。

## 我们为什么需要你的帮助

一个人或几个人的力量终究有限，而《无期迷途》的世界观、角色机制与关卡设计非常丰富。为了让 Wiki 真正做到“全、准、新”，我们诚挚邀请你加入共建：

- **角色资料**：补充角色背景、技能解析、烙印推荐、使用场景；
- **关卡攻略**：撰写主线、破碎防线、活动关卡的打法与阵容；
- **数据校对**：发现技能数值、TAG、描述错误时及时反馈；
- **美术与排版**：优化页面视觉、图标、截图规范；
- **程序与工具**：改进抓取脚本、搜索体验、自动化流程。

## 如何参与

### 1. 通过 GitHub 提交 Pull Request（推荐）

- 仓库地址：https://github.com/vajhXajhcv/ptn-wiki
- 角色内容位于 `src/content/characters/`；
- 关卡内容位于 `src/content/stages/`；
- 修改后运行 `npm run build` 确保站点能正常构建；
- 提交 PR 时简要说明改动原因与依据（如游戏内实测、官方公告、BWiki 等）。

### 2. 提交 Issue 反馈

如果你不方便直接改代码，也欢迎在 GitHub Issues 中提交：

- 错误反馈（错别字、数值错误、失效链接等）；
- 内容建议（缺少的角色、关卡、攻略主题）；
- 功能建议（搜索优化、筛选器、新页面）。

### 3. 加入讨论

我们会在仓库 README 与站点底部留下长期有效的交流方式，欢迎围绕 Wiki 内容、规范与工具展开讨论。

## 共建原则

1. **非营利**：本站不投放广告、不接受付费推广、不出售任何虚拟或实体商品。
2. **尊重版权**：角色图片使用官网公开素材；数据参考 BWiki 等社区公开整理；原创内容归贡献者所有，以 MIT License 开源授权。
3. **有据可查**：攻略与数据尽量给出来源或实测依据，避免主观夸大。
4. **友好协作**：尊重每一位贡献者，讨论就事论事，共同维护良好的社区氛围。

## 你能获得什么

- 你贡献的内容将出现在 https://5732.wiki，帮助更多局长；
- 你的 GitHub ID 会记录在提交历史中，作为开源贡献的证明；
- 与一群同样热爱《无期迷途》的玩家一起，打造一个真正属于社区的资料站。

无论你是大佬还是萌新，只要愿意分享一点点时间或知识，就是对 Wiki 最大的支持。
