## 玩法概述

狄斯暗影提供一系列高难度关卡，敌人拥有特殊机制与强化属性。适合主力成型后挑战，获取稀有奖励。

## 核心机制

- **特殊机制**：每关可能有独特胜利条件或敌人词缀。
- **高难度**：敌人等级与属性较高，需要完整配队。
- **一次性奖励**：首通奖励丰厚，部分可重复挑战。
- **阶段性解锁**：通关前置关卡后解锁后续更高难度关卡。

## 通用挑战思路

1. **阅读关卡机制**：每关的特殊条件往往决定胜负，比如限制职业、保护目标、限时击杀等。
2. **准备完整配队**：建议带上破核、治疗、增益辅助和主 C，应对复杂局面。
3. **分阶段熟悉**：如果一次无法通关，可以多次尝试，熟悉敌人刷新位置与技能释放时机。
4. **调整烙印与技能**：针对关卡特性微调烙印套装，比如需要爆发时带增伤套，需要生存时带防御套。

## 常见关卡类型

| 类型 | 要点 |
|------|------|
| 限时击杀 | 优先清理精英，集中爆发 |
| 保护目标 | 放置坚韧/狂暴角色吸引仇恨 |
| 生存挑战 | 带足治疗与护盾，合理利用地形 |
| 职业限制 | 提前培养各职业角色 |
