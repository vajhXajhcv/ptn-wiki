尊敬的无期迷途 BWiki 维护团队：

您好！

我们是《无期迷途》玩家 Wiki **5732.wiki** 的维护团队。本站是一个面向全体局长的中文资料站，采用 Astro 构建静态站点，源码以 MIT License 开源托管于 GitHub。我们在整理角色技能、特性、机制说明的过程中，参考了贵站公开的 Wikitext 数据，并在此基础上进行了格式转换与内容重组。

## 参考方式说明

我们通过 BWiki MediaWiki API 读取 `禁闭者:角色名` 页面源码，提取 `{{禁闭者图鉴}}` 模板中的以下公开字段：

- 角色特性、狂厄深化、梦魇天赋；
- 普攻、必杀、被动 1、被动 2 的名称与效果；
- 技能范围、能量消耗、核心伤害等客观数值。

在引用过程中，我们会将 `{{伤害文本}}`、`{{禁闭者数值}}`、`{{技能范围}}`、`{{术语查看}}` 等模板转换为 Markdown 可读文本，并重新组织成“特性与机制”“技能”“使用建议”“烙印推荐”等章节，而非直接复制贵站页面。

## 我们的原则

1. **非营利**：本站不投放广告、不售卖商品、不参与任何商业变现。
2. **显著致谢**：我们已在 `docs/official-assets.md` 与站点相关文档中明确标注技能数据参考自无期迷途 BWiki。
3. **尊重原创**：对于攻略、使用建议等主观内容，均由我们原创整理；对于贵站社区原创的分析、测评等内容，我们不会直接搬运。
4. **可沟通下架**：若贵站对任何引用方式存在异议，或希望我们补充更明确的署名、链接，我们将在第一时间配合调整。

## 期待的合作

我们非常敬佩 BWiki 团队长期以来对《无期迷途》资料的系统性整理。若贵站愿意，我们希望能建立一种良性的社区协作关系：

- 在明显位置相互标注来源与链接，方便玩家在不同站点间跳转；
- 在发现数据错误时互相反馈、共同校对；
- 探讨更规范的数据引用与开源协作方式，避免重复劳动。

再次感谢贵站为《无期迷途》玩家社区做出的贡献。期待能与你们保持沟通，共同为广大局长提供更好的资料服务。
